# Delivery APP

## Description

This project is deliveryapp challenge mobile application that is developed in react-native using TypeScript.

## Requirements

- Android SDK
- Xcode
- xcode-tools

## Setup

```shell
  yarn install
  bundle update
  npx pod-install
```

## Run storybook
```
yarn storybook
```
